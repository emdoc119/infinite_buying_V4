무한매수법 V4.0 핵심 가이드

1. 공통 원칙

  • 종목은 TQQQ / SOXL 한정. 타 종목은 오피셜이 아닙니다.
  • 모든 주문은 LOC (Limit on Close) 기반으로, 프리장 시작 직후에 주문을 걸어둡니다. (주간장 이용 금지)
  • 여러 종목 / 분할 방식 병행 시 원금은 철저히 독립적으로 설정해야 합니다.

2. 주요 용어

  • T (Turn, 회차): 진행 경과를 나타내는 값입니다. 1회 매수 시 +1, 절반 매수 시 +0.5 가산됩니다.
  • 별지점 / 별% (☆%): 매수와 매도를 가르는 기준 퍼센트 (평단 대비) 입니다. TQQQ 40분할 기준 (15 - 0.75T)% 공식이 적용됩니다.
  • LOC 매수/매도: Limit on Close. 종가가 기준 이하(매수) 또는 이상(매도)일 때만 종가로 체결되는 주문 방식입니다.
  • 쿼터매수: 리버스 모드에서 방어적으로 평단을 낮추는 행위입니다.

3. 리버스 모드 (V4.0)

원금이 모두 소진되었을 때 자동으로 돌입하는 방어 모드입니다. 전체 수량의 1/20(또는 1/10)을 무조건 매도(MOC)하고, 매도된 금액으로 다시 쿼터매수하여 평단을 공격적으로 낮춥니다. 평단 대비 손실률이 -15%(TQQQ) 위로 올라오면 다시 일반 모드로 복귀합니다.
